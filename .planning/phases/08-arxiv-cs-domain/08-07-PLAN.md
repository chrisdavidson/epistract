<objective>
Add arXiv abstract URL (`abs_url`) to every PAPER node in the knowledge graph so users can navigate directly from a node popup to the source paper. The URL is 100% deterministic from the arXiv ID — no re-extraction is needed.

Purpose: PAPER nodes currently have `attributes: {}`. The arXiv ID is already the node name (e.g., "1706.03762"), so the canonical URL `https://arxiv.org/abs/{arxiv_id}` can be injected by the epistemic enrichment step on every rebuild. The workbench displays node attributes in the node popup, making the URL visible and clickable without any UI changes.

Output: Updated epistemic.py (URL enrichment loop), updated SKILL.md (Metadata Attributes section), 8 patched extraction JSONs (abs_url on PAPER entities), rebuilt graph_data.json with abs_url on all PAPER nodes, one new unit test.
</objective>

    # --- URL enrichment: inject abs_url onto PAPER nodes ---
    # The arXiv ID is already the node name (e.g. "1706.03762").
    # The abs_url is 100% deterministic — no LLM extraction needed.
    # Pattern guard r"^\d+\.\d+$" accepts arXiv IDs and rejects all
    # other entity type names (algorithms, datasets, authors, etc.).
    # This runs on every label_epistemic invocation so graph rebuilds
    # always carry the URL without requiring re-extraction.
    import re as _re

    _ARXIV_ID_RE = _re.compile(r"^\d+\.\d+$")
    for node in nodes:
        if node.get("entity_type") == "PAPER":
            name = node.get("name", "")
            if _ARXIV_ID_RE.match(name):
                if not node.get("attributes"):
                    node["attributes"] = {}
                node["attributes"]["abs_url"] = f"https://arxiv.org/abs/{name}"
```

import json
import re
from pathlib import Path

EXTRACTIONS_DIR = Path("tests/corpora/09_arxiv_cs/output/extractions")
ARXIV_ID_RE = re.compile(r"^\d+\.\d+$")

for json_path in sorted(EXTRACTIONS_DIR.glob("*.json")):
    if json_path.name.startswith("_"):
        continue  # skip _normalization_report.json
    data = json.loads(json_path.read_text(encoding="utf-8"))
    modified = False
    for entity in data.get("entities", []):
        if entity.get("entity_type") == "PAPER":
            name = entity.get("name", "")
            if ARXIV_ID_RE.match(name):
                if not entity.get("attributes"):
                    entity["attributes"] = {}
                entity["attributes"]["abs_url"] = f"https://arxiv.org/abs/{name}"
                modified = True
    if modified:
        json_path.write_text(json.dumps(data, indent=2) + "\n", encoding="utf-8")
        print(f"Patched: {json_path.name}")
    else:
        print(f"Skipped (no matching PAPER entity): {json_path.name}")
EOF
```
