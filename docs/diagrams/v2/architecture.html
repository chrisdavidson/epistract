Epistract v2.0 — System Architecture

Domain-agnostic knowledge graph framework · three-layer separation · plug-in domain packages

Three-Layer Principle

  • • core/ imports nothing domain-specific
  • • domains/ = domain.yaml + SKILL.md + epistemic.py
  • • examples/ = reference consumers, swappable
  • • Adding a domain = zero core code changes

Domain Packages (v2.0)

  • • drug-discovery — 17 entities · 30 relations · RDKit + Biopython
  • • contracts — 11 entities · 11 relations · severity layer
  • • /epistract:domain wizard scaffolds new packages from sample corpora
  • • AST-validated epistemic.py generation

External Stack

  • • sift-kg ≥0.9.0 — MultiDiGraph + community detection
  • • kreuzberg ≥4.0 — 75+ formats + Tesseract OCR
  • • 3 LLM providers pluggable via env vars
  • • Optional: RDKit · Biopython · Tesseract · PubMed E-utils
