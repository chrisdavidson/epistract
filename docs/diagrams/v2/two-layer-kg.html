Epistract v2.0 — Two-Layer Knowledge Graph

Layer 1 captures what documents say · Layer 2 captures what we infer about those facts (asserted · hypothesized · prophetic · contradictory · gap)

Layer 1 — Brute Facts

  • • Entities + relations extracted verbatim from documents
  • • Domain schema constrains permitted types
  • • Nodes deduped via canonical name + SemHash fuzzy match
  • • Queryable via /epistract:query + /epistract:view
  • • Same graph-build code runs for every domain

Layer 2 — Epistemic Claims

  • • Asserted — high-confidence factual claims (multiple sources agree)
  • • Hypothesized — hedging language detected in source text
  • • Prophetic — patent-sourced, not peer-reviewed
  • • Contradictory — cross-document conflicts flagged
  • • Gap — expected relations absent from corpus

Domain-Pluggable Layer 2

  • • drug-discovery — 4 statuses + multi-hypothesis detection
  • • contracts — Critical · Warning · Info severity instead
  • • New domains plug in epistemic.py with custom rule set
  • • core/epistemic.py dispatches via DOMAIN_ALIASES + getattr
  • • Workbench renders both Layer 1 + Layer 2 as overlays
